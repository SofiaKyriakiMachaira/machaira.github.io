# machaira.github.io
A Javascript resume template project / sandbox currently presenting my resume.

- Provides easy-to-use code to display a resume, including space for extra details if needed.
- Suggests a safer way to display contact info against spam callers & emails.
- Colours can also be easily edited. It is advised to use a colour contrast accessibility checker.

Technologies used:
HTML, CSS, Javascript.

Might be re-developed or enhanced in future commits.
